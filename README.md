This repository contains the code, notebooks, and materials for the master’s thesis focused on developing an automated pipeline for analyzing semi-structured text from Electronic Medical Records (EMRs) to support personalized clinical decision-making.

Files discription:</br>

MethodsComparing - a file comparing different approaches to dividing text entries into sentences.</br>

ResearchWorkPreprocessingAlmazovRazdel - a file with pre-processing of the main text field with general recommendations from the doctor.</br>

ResearchWorkAlmazovClusteringRazdel - a file with the basic algorithm for working with already processed text data. At this stage, topic modeling is performed, and then records within each topic are clustered using the DBSCAN algorithm.</br>
